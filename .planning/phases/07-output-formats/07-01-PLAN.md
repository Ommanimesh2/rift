<feature>
  <name>JSON output formatter</name>
  <files>internal/output/json.go, internal/output/json_test.go</files>
  <behavior>
    FormatJSON(result *diff.DiffResult, image1, image2 string, events []security.SecurityEvent) ([]byte, error)

    Top-level struct (DiffReport):
    {
      "image1": "nginx:1.24",
      "image2": "nginx:1.25",
      "summary": {
        "added": 2,
        "removed": 1,
        "modified": 3,
        "added_bytes": 12345,
        "removed_bytes": 4567
      },
      "changes": [...],
      "security_events": [...]
    }

    Change entry (ChangeEntry):
    - Added:    { "path": "usr/bin/wget", "type": "added",    "size_delta": 123456 }
    - Removed:  { "path": "usr/lib/old.so", "type": "removed",  "size_delta": -89012 }
    - Modified with flags:
                { "path": "etc/nginx.conf", "type": "modified", "size_delta": 45,
                  "changes": ["content", "mode"] }
    - Modified no flags: "changes" key is omitted (json:"changes,omitempty")

    Security entry (SecurityEntry):
    - Added SUID (Before==0):    { "kind": "new_suid", "path": "/usr/bin/ping", "before_mode": 0, "after_mode": 2541 }
    - Modified perm escalation:  { "kind": "perm_escalation", "path": "/bin/sh", "before_mode": 493, "after_mode": 2541 }
      (modes stored as uint32 integer, e.g. 0755 octal = 493 decimal)

    Empty diff result:
    - summary with all-zero counts, "changes": [], "security_events": []

    Nil events slice:
    - "security_events": [] (not null)

    Output is json.MarshalIndent with "" prefix and "  " (2-space) indent.

    Function signature and struct names:
    - package output
    - type DiffReport struct { Image1, Image2, Summary, Changes, SecurityEvents }
    - type ReportSummary struct { Added, Removed, Modified, AddedBytes, RemovedBytes }
    - type ChangeEntry struct { Path, Type, SizeDelta, Changes ([]string, omitempty) }
    - type SecurityEntry struct { Kind, Path, BeforeMode, AfterMode (uint32) }

    Change flags in "changes" array follow same canonical order as collectFlags:
    content, mode, uid, gid, link, type
  </behavior>
  <implementation>
    Create internal/output/json.go in package output.

    Import: encoding/json, github.com/ommmishra/imgdiff/internal/diff, github.com/ommmishra/imgdiff/internal/security

    Define the struct types with json tags (snake_case). For SecurityEntry.Kind use string(event.Kind) to serialize the SecurityEventKind string type naturally.

    FormatJSON builds a DiffReport, converts each DiffEntry to a ChangeEntry (call collectFlags for Modified), converts each SecurityEvent to a SecurityEntry (cast os.FileMode to uint32), then calls json.MarshalIndent.

    Do NOT add a trailing newline — return the raw bytes from json.MarshalIndent.

    The "changes" field on ChangeEntry should only be populated for Modified entries; Added/Removed entries leave it nil (omitempty handles the rest).

    For nil events, initialize SecurityEvents as []SecurityEntry{} (not nil) so JSON renders [] not null.
  </implementation>
</feature>
