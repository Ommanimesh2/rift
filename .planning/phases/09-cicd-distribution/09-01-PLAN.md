  <behavior>
Package `exitcode` provides two exported functions:

**ParseSizeThreshold(s string) (int64, error)**
Parse a human-readable size string to bytes. Supported suffixes (case-insensitive):
- "" or "0" → 0, nil (disabled)
- "100" → 100, nil (bare number = bytes)
- "500B" → 500, nil
- "10KB" or "10K" → 10240, nil
- "5MB" or "5M" → 5242880, nil
- "1GB" or "1G" → 1073741824, nil
- Decimal values: "1.5MB" → 1572864, nil
- Negative values → error
- Unknown suffix → error
- Non-numeric → error

**type Options struct**
- ExitOnChange bool   — trigger if any file added/removed/modified
- ExitOnSecurity bool — trigger if any security events
- SizeThreshold int64 — trigger if net size delta (AddedBytes - RemovedBytes) > threshold; 0 = disabled

**Evaluate(result *diff.DiffResult, events []security.SecurityEvent, opts Options) int**
Returns:
- 0 — no conditions triggered
- 2 — one or more conditions triggered (2 matches `diff` command convention for "differences found")

Trigger rules (any one is sufficient to return 2):
- ExitOnChange && (result.Added + result.Removed + result.Modified > 0) → 2
- ExitOnSecurity && len(events) > 0 → 2
- SizeThreshold > 0 && (result.AddedBytes - result.RemovedBytes) > SizeThreshold → 2

Test cases for Evaluate:
- empty result + no events + all options false → 0
- result with changes + ExitOnChange=true → 2
- result with changes + ExitOnChange=false → 0 (change doesn't matter)
- no changes + ExitOnChange=true → 0 (nothing to trigger)
- events present + ExitOnSecurity=true → 2
- events present + ExitOnSecurity=false → 0
- net delta = 5MB + SizeThreshold = 1MB → 2
- net delta = 500KB + SizeThreshold = 1MB → 0 (under threshold)
- SizeThreshold = 0 (disabled) + large delta → 0
- all three options true + no changes/events → 0
- net delta negative (image shrank) + SizeThreshold > 0 → 0 (only triggers on growth)
  </behavior>

```go
// Options configures which conditions trigger a non-zero exit code.
type Options struct {
    ExitOnChange   bool
    ExitOnSecurity bool
    SizeThreshold  int64 // bytes; 0 = disabled
}

// ParseSizeThreshold parses a human-readable size string to bytes.
// Supported: bare number (bytes), B, KB/K, MB/M, GB/G (case-insensitive).
// Decimal values are supported (e.g., "1.5MB").
func ParseSizeThreshold(s string) (int64, error) { ... }

// Evaluate checks whether any configured condition is triggered.
// Returns 0 if no conditions are triggered, 2 if any condition is triggered.
// Exit code 2 follows the `diff` command convention for "differences found".
func Evaluate(result *diff.DiffResult, events []security.SecurityEvent, opts Options) int { ... }
```

For ParseSizeThreshold:
- strings.TrimSpace, then handle empty/zero
- Extract numeric prefix with strconv.ParseFloat
- Extract suffix, normalize to uppercase
- Apply multiplier (K=1024, M=1024², G=1024³, B=1)
- Return int64(value * multiplier), error if negative

For Evaluate:
- Check each condition independently
- Return 2 at first triggered condition (short-circuit)
- Net delta = result.AddedBytes - result.RemovedBytes (only trigger on positive delta)
