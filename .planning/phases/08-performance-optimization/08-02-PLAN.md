<feature>
  <name>Quick-mode output formatter (format-aware)</name>
  <files>internal/output/quick.go, internal/output/quick_test.go</files>
  <behavior>
    FormatQuick(summary *LayerSummary, image1, image2, format string) string

    Handles three format values:

    format == "terminal" or "":
      Plain-text output (no lipgloss dependency — keeps quick.go simple):
        Quick comparison: {image1} → {image2}

        {FormatLayerSummary output}

        (No file content downloaded — run without --quick for file-level diff)

    format == "json":
      Indented JSON object (2-space):
        {
          "image1": "nginx:1.24",
          "image2": "nginx:1.25",
          "mode": "quick",
          "layers": {
            "total_a": 5,
            "total_b": 6,
            "shared": 4,
            "shared_bytes": 524288000,
            "only_in_a": 1,
            "only_in_a_bytes": 10240,
            "only_in_b": 2,
            "only_in_b_bytes": 20480
          }
        }
      Include trailing newline.

    format == "markdown":
      ## Quick Image Comparison: `{image1}` → `{image2}`

      > Manifest-only comparison — no file content downloaded.

      {FormatLayerSummary output in a fenced code block}

    Nil summary handling:
      If summary is nil, all layer fields are zero. Must not panic in any branch.

    Unknown format:
      Return fmt.Sprintf("unsupported format: %s", format)

    Test cases:
    - FormatQuick(summary, "img1", "img2", "terminal") contains both image names
    - FormatQuick(summary, "img1", "img2", "terminal") contains shared count
    - FormatQuick(summary, "img1", "img2", "json") is valid JSON (unmarshal check)
    - FormatQuick(summary, "img1", "img2", "json") → image1 field == "img1"
    - FormatQuick(summary, "img1", "img2", "json") → mode field == "quick"
    - FormatQuick(summary, "img1", "img2", "json") → layers.shared == expected
    - FormatQuick(summary, "img1", "img2", "markdown") contains "## Quick Image Comparison"
    - FormatQuick(summary, "img1", "img2", "markdown") contains image names
    - FormatQuick(nil, "img1", "img2", "terminal") does not panic
    - FormatQuick(nil, "img1", "img2", "json") does not panic, valid JSON
    - FormatQuick(summary, "img1", "img2", "") treated same as "terminal"
    - FormatQuick(summary, "img1", "img2", "unknown") returns error string
  </behavior>

    - default: return unsupported format string
