<task type="auto">
  <name>Task 2: Check size and optimize if needed</name>
  <files>demo.gif</files>
  <action>
Check file size and optimize if over 3MB:

```bash
SIZE=$(du -k demo.gif | cut -f1)
echo "demo.gif size: ${SIZE}KB ($(( SIZE / 1024 ))MB)"

if [ "$SIZE" -gt 3072 ]; then
  echo "GIF is over 3MB — optimizing with gifsicle..."
  brew install gifsicle 2>/dev/null || true

  # Try O3 optimization first
  gifsicle -O3 demo.gif -o demo-opt.gif
  SIZE_OPT=$(du -k demo-opt.gif | cut -f1)
  echo "After O3: ${SIZE_OPT}KB"

  if [ "$SIZE_OPT" -lt 3072 ]; then
    mv demo-opt.gif demo.gif
    echo "Replaced with optimized version"
  else
    # Reduce colors
    gifsicle -O3 --colors 128 demo.gif -o demo-opt.gif
    SIZE_OPT2=$(du -k demo-opt.gif | cut -f1)
    echo "After color reduction: ${SIZE_OPT2}KB"
    mv demo-opt.gif demo.gif
    echo "Replaced with color-reduced version"
  fi
else
  echo "GIF is under 3MB — no optimization needed"
fi

# Final size check
echo "Final size: $(du -h demo.gif | cut -f1)"
```

If GIF is still over 3MB after optimization, reduce dimensions in demo.tape:
- Change `Set Width 1000` → `Set Width 900` and re-render
- Or reduce `Set Height 620` → `Set Height 560`
Re-run vhs if dimensions changed.
  </action>
