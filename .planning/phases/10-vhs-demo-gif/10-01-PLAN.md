<task type="auto">
  <name>Task 2: Write demo.tape script</name>
  <files>demo.tape</files>
  <action>
Create `demo.tape` in the project root. The tape must:
- Pre-pull images under Hide/Show to eliminate network wait from the visible GIF
- Show 3 compelling scenes in sequence
- Use Catppuccin Mocha theme (matches charmbracelet ecosystem)
- Stay under 60 seconds total

**Critical tape rules from research:**
- ALL Set/Output commands MUST come before any Type/Enter/interaction
- Quote all Type strings
- Use `Wait+Screen` with generous timeout for imgdiff (it pulls from registry)
- Shell prompt pattern for zsh: use `Wait+Screen /\$ /` or a known string

Write the following tape:

```tape
Output demo.gif

Set FontSize 14
Set Width 1000
Set Height 620
Set Theme "Catppuccin Mocha"
Set Padding 20
Set TypingSpeed 80ms
Set Shell "zsh"
Set WindowBar Colorful

# ── Pre-pull images silently ──────────────────────────────────────────────────
Hide
Type "./imgdiff nginx:1.24 nginx:1.25 --format json > /dev/null 2>&1; echo DONE"
Enter
Wait+Screen /DONE/
Wait 1
Type "./imgdiff ubuntu:22.04 ubuntu:24.04 --format json > /dev/null 2>&1; echo DONE"
Enter
Wait+Screen /DONE/
Wait 1
Show

# ── Clear terminal for clean start ────────────────────────────────────────────
Type "clear"
Enter
Wait 0.5

# ── Scene 1: File diff — nginx upgrade ────────────────────────────────────────
Type "# Scene 1: What changed in nginx 1.24 → 1.25?"
Enter
Sleep 1
Type "./imgdiff nginx:1.24 nginx:1.25"
Enter
Wait+Screen /Total files/
Sleep 4

# ── Scene 2: Security — ubuntu major upgrade ──────────────────────────────────
Type "clear"
Enter
Wait 0.3
Type "# Scene 2: Security changes in ubuntu 22.04 → 24.04"
Enter
Sleep 1
Type "./imgdiff ubuntu:22.04 ubuntu:24.04"
Enter
Wait+Screen /Total files/
Sleep 4

# ── Scene 3: JSON for CI/CD ───────────────────────────────────────────────────
Type "clear"
Enter
Wait 0.3
Type "# Scene 3: JSON output for CI/CD pipelines"
Enter
Sleep 1
Type "./imgdiff nginx:1.24 nginx:1.25 --format json | head -30"
Enter
Wait+Screen /}/
Sleep 3
```

Write this exactly to `demo.tape` in the project root.

**Note on Wait+Screen pattern:** `Total files` should appear in imgdiff output. If imgdiff terminal output uses a different summary line, adjust the Wait pattern after checking `./imgdiff nginx:1.24 nginx:1.25 2>&1 | tail -5`.

Check the actual output summary line first:
```bash
./imgdiff --help 2>&1 | grep -i "total\|summary\|files" || echo "check actual run"
```

Adjust `Wait+Screen` patterns in demo.tape to match actual output if needed.
