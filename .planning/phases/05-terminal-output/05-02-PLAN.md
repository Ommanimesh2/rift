<task type="auto">
  <name>Task 1: Add lipgloss dependency and create terminal renderer</name>
  <files>go.mod, go.sum, internal/output/terminal.go</files>
  <action>
    Run `go get github.com/charmbracelet/lipgloss@latest` to add the dependency.

    Create internal/output/terminal.go with a `RenderTerminal(result *diff.DiffResult, image1Name, image2Name string) string` function that:

    1. Defines lipgloss styles:
       - addedStyle: green foreground (lipgloss.Color("2") or similar ANSI green)
       - removedStyle: red foreground (lipgloss.Color("1") or ANSI red)
       - modifiedStyle: yellow foreground (lipgloss.Color("3") or ANSI yellow)
       - pathStyle: bold
       - headerStyle: bold + underline
       - dimStyle: faint/dim for metadata like change flags and size deltas

    2. Renders a header section:
       - "Comparing {image1Name} → {image2Name}" with headerStyle
       - Blank line

    3. Renders per-entry lines using the styles:
       - Added: green "+" prefix, path in green+bold, size delta in dim
       - Removed: red "-" prefix, path in red+bold, size delta in dim
       - Modified: yellow "~" prefix, path in yellow+bold, change flags in dim brackets, size delta in dim
       - Use FormatSizeDelta from format.go for the delta text

    4. Renders a summary section:
       - Blank line separator
       - "Summary: {N} added (+{bytes}), {N} removed (-{bytes}), {N} modified"
       - Color each count with its corresponding style (added count in green, removed in red, modified in yellow)

    5. Handles empty diff: "No differences found." with dim style

    Use lipgloss.NewStyle() builder pattern. Do NOT use lipgloss.Render() deprecated API — use style.Render(text) method instead.
    Keep ANSI color codes in output (lipgloss handles terminal detection automatically).
  </action>
  <verify>go build ./... compiles successfully with lipgloss imported</verify>
  <done>internal/output/terminal.go exists with RenderTerminal function, project compiles with lipgloss dependency</done>
</task>
