<feature>
  <name>Layer comparison and breakdown</name>
  <files>internal/output/layers.go, internal/output/layers_test.go</files>
  <behavior>
    LayerSummary type:
    - SharedCount int       — layers with matching DiffID in both images
    - SharedBytes int64     — combined compressed size of shared layers
    - OnlyInACount int      — layers unique to image A
    - OnlyInABytes int64    — combined compressed size of unique-to-A layers
    - OnlyInBCount int      — layers unique to image B
    - OnlyInBBytes int64    — combined compressed size of unique-to-B layers
    - TotalA int            — total layers in image A
    - TotalB int            — total layers in image B

    CompareLayers(imgA, imgB v1.Image) (*LayerSummary, error):
    - Get layers from both images via img.Layers()
    - For each layer, get DiffID (digest of uncompressed content) via layer.DiffID()
    - Collect DiffID sets for A and B
    - Shared = intersection (DiffID appears in both)
    - OnlyInA = A - shared
    - OnlyInB = B - shared
    - For sizes, use layer.Size() (compressed size) for each layer
    - Return LayerSummary with all counts and sizes

    Test cases:
    - Identical images (all layers shared, no unique)
    - Completely different images (no shared layers)
    - Partially shared (some shared, some unique to each)
    - Empty image A, non-empty image B
    - Single layer images

    FormatLayerSummary(summary *LayerSummary) string:
    - "Layers: {totalA} → {totalB}"
    - "  Shared:  {sharedCount} layers ({sharedBytes})"
    - "  Only in A: {onlyInACount} layers ({onlyInABytes})"  — omit line if 0
    - "  Only in B: {onlyInBCount} layers ({onlyInBBytes})"  — omit line if 0
    - Use FormatBytes for sizes

    Test FormatLayerSummary with various LayerSummary inputs.

    Use a fakeImage test helper implementing v1.Image that returns configurable fake layers with known DiffIDs and sizes. Use the same fakeLayer pattern from tree package tests — implement v1.Layer with predetermined DiffID and Size.
  </behavior>
  <implementation>
    Create internal/output/layers.go with CompareLayers and FormatLayerSummary.

    CompareLayers algorithm:
    1. Get layers from both images
    2. Build map[v1.Hash]v1.Layer for image A layers (keyed by DiffID)
    3. Build map[v1.Hash]v1.Layer for image B layers (keyed by DiffID)
    4. Iterate A map: if DiffID in B map → shared, else → onlyInA
    5. Iterate B map: if DiffID not in A map → onlyInB
    6. Sum sizes using layer.Size() for each category
    7. Return populated LayerSummary

    FormatLayerSummary: simple string builder with conditional lines.

    For tests: create fakeImage and fakeLayer types that implement v1.Image and v1.Layer interfaces respectively. fakeLayer needs DiffID() and Size() methods. fakeImage needs Layers() method returning slice of fakeLayer. Other v1.Image/v1.Layer methods can panic("not implemented").

    After TDD is complete (tests pass), wire into cmd/root.go:
    - After computing diff result, call output.CompareLayers(img1, img2)
    - Pass LayerSummary to a new RenderLayerSection function (or add to RenderTerminal)
    - Display layer breakdown between the header and file listing
    - Use lipgloss dim style for the layer section (it's supplementary info)
    - If CompareLayers returns error, log warning and skip layer section (don't fail the command)
  </implementation>
</feature>
