<objective>
Wire BuildTree into the CLI by adding a BuildFromImage convenience function and updating the root command to build file trees and output summary statistics.

Purpose: Connect the tree engine to the existing CLI so that running `imgdiff image1 image2` builds file trees from both images. This replaces the current metadata-only output with tree statistics (file count, total size, directory count) as a stepping stone toward the diff output in Phase 4.
Output: CLI that opens images, builds squashed file trees, and prints tree summary for each image.
